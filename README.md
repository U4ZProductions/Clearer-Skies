# Clearer-Skies
The official open source sequel to TRACE_AQI2.0
                               Clearer-Skies
Our developer at Clear-Skies / U4ZProductions has developed an application to view and register factors that affect global warming globally or locally. 

Our program works with simple latitude and longitude entries and has preset countries for the convenience of casual users.

Students and academists looking to investigate local areas for a project about global warming but users can always look up coordinates and then input them to use our program to it’s max potential and find natural causes, organizations and elements that cause major harm.

We have the aim of shining light to the people, elements that are ignored when made but then regretted because of the harm it causes to natural habitats and global warming “Sera Etkisi”

We have used  open non-commercial Open-Meteo API’s to get real time data and experimented with NASA Open API subsystems to comprehend and most importantly compile data for our program
